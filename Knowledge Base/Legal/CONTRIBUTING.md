# Contributing to CoSyn

Thank you for your interest in contributing.

## Contributor License

By submitting contributions you agree that:

- Your contribution is your original work or appropriately licensed.
- You grant the project maintainer a perpetual, worldwide license to use, modify, and distribute the contribution.

## Governance Compliance

All contributions must comply with:

- CoSyn Constitution
- Persona Governor
- Stack Architect architecture constraints

## Code Standards

Contributions should:

- Maintain deterministic behavior
- Preserve governance gate integrity
- Avoid bypassing governance enforcement logic

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Submit a pull request
4. Include description and rationale

## Review

All contributions are subject to maintainers' review for:

- security
- architectural consistency
- governance compliance
